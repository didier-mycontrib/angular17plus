Bases de typescript

logiciels à télécharger et installer si besoin:

            * nodeJs (choisir dernière version LTS)
            * VisualStudioCode (bon IDE pour javascript/typescript/angular)

Premiers pas avec typescript:

            Vérification de l'installation de node/npm:
            ------------------------------------------
            node --version
            npm --version

            Installation de typescript et vérification:
            ------------------------------------------
            npm install -g typescript
            tsc --version

            Au sein du répertoire d'un projet (ex: "rappels-typescript"):
            ------------------------------------------
            * visualiser le contenu du fichier src/hello-world.ts
            * se placer dans le répertoire src (au sein d'un terminal)
            * lancer la commande suivante:
                 tsc hello-world.ts
            * visualiser le fichier src/hello-world.js construit
            * lancer la commande suivante:
               node hello-world.js
            * supprimer le fichier src/hello-world.js (car pas à l'entroit idéal)

contenu de hello-world.ts:

            let message : string ;
            //message = 5; //wrong type
            message = "hello world";
            console.log(message);

            const a =8;
            console.log("a="+a);
