Configuration de typescript-

Création d'un fichier tsconfig.json:

            Au sein du répertoire d'un projet (ex: "rappels-typescript"):
            ------------------------------------------
            tsc --init

Contenu important de tsconfig.json

            {
                "compilerOptions": {
                  "target": "es2017",
                  "module" : "CommonJS",
                  "rootDir": "src", 
                  "outDir": "dist/js",
                  "strict": true  
                }
            }

Utilisation de typescript en mode "projet":

          
            Au sein du répertoire d'un projet (ex: "rappels-typescript"):
            ------------------------------------------
            * supprimer tout les fichiers du répertoire "dist/js" (qui seront reconstruits)
            * se placer à la racine du projet (au sein d'un terminal)
            * lancer la commande suivante:
                 tsc 
            * visualiser les fichiers ".js" construits au sein de dist/js
            * lancer la commande suivante:
                  node dist/js/hello-world.js
            * effectuer facultativement d'autres petits tests
